Beer'd

Beer'd is a beer indentifying app with many different features and API uses.

It uses three properties to produce three different types of beers that match.

You can then share on facebook, re-search, or find the closest LCBO location.

Afterwards, it stores the search, up to three previous searches so you can access it again.
